# Portfolio Akuntansi (Next.js + Supabase)

Website portofolio akuntansi dengan:

- halaman publik (profil, skill, pengalaman, CV ATS)
- dashboard tersembunyi di `/dashboard` untuk pemilik
- integrasi Supabase untuk simpan data
- generator CV ATS simple yang bisa diedit
- export CV ke PDF (`/api/portfolio/cv/pdf`)

## 1) Menjalankan project

```bash
npm install
npm run dev
```

Lalu buka `http://localhost:3000`.

## 2) Environment variable

Buat file `.env.local`:

```env
SUPABASE_URL=https://YOUR_PROJECT.supabase.co
SUPABASE_SERVICE_ROLE_KEY=YOUR_SERVICE_ROLE_KEY
OWNER_DASHBOARD_KEY=gantidengankuncirahasia # opsional (default pakai SUPABASE_SERVICE_ROLE_KEY)
```

> Jika variabel Supabase belum diisi, aplikasi otomatis pakai fallback data lokal (in-memory).

## 3) Struktur tabel Supabase

Gunakan SQL ini di Supabase SQL Editor:

```sql
create table if not exists portfolio_profile (
  id bigint primary key,
  full_name text not null,
  headline text not null,
  bio text not null,
  email text not null,
  location text not null,
  cv_content text not null default ''
);

create table if not exists portfolio_skills (
  id bigint generated by default as identity primary key,
  name text not null,
  level text not null,
  sort_order int not null
);

create table if not exists portfolio_experiences (
  id bigint generated by default as identity primary key,
  role text not null,
  company text not null,
  period text not null,
  description text not null,
  sort_order int not null
);
```

## 4) Dashboard tersembunyi

- akses langsung: `http://localhost:3000/dashboard`
- masukkan owner key (`OWNER_DASHBOARD_KEY`) di form awal
- update profil, skill, pengalaman, dan CV
- klik `Generate ATS` untuk generate CV otomatis dari data terbaru
- klik `Export PDF` untuk unduh CV PDF
